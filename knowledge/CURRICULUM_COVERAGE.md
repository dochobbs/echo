## YAML Template for New Conditions

Each condition YAML should include:

```yaml
# [Condition Name] - Echo Teaching Condition
# Adapted from Master Pediatric Primary Care Curriculum

display_name: "Condition Name"
aliases: ["Alternative names"]

billing_codes:
  icd10: ["code"]
  snomed: "code"

category: acute|chronic|developmental
system: respiratory|gi|derm|etc

demographics:
  age_months: { min: X, peak: [X, X], max: X }
  gender_bias: { male: 0.5, female: 0.5 }
  risk_factors: ["list"]

seasonality:
  peak_months: [1, 2, 3]  # if applicable

vitals_impact:
  temp_f: [98.6, 98.6]
  hr_multiplier: 1.0
  rr_multiplier: 1.0
  spo2_min: 98

presentation:
  chief_complaints:
    - "Parent quote 1"
    - "Parent quote 2"
  symptoms:
    - "symptom1"
    - "symptom2"
  duration_days: [X, X]
  physical_exam:
    - { system: "general", finding: "Finding description" }

treatment:
  approach: "Overall approach"
  medications:
    - agent: "Drug name"
      dose_mg_kg: X
      max_dose_mg: X
      frequency: "dosing"
  supportive:
    - "Supportive measure 1"

# ============= TEACHING ENHANCEMENTS =============

teaching_goals:
  - "Goal 1 from curriculum"
  - "Goal 2"

common_learner_mistakes:
  - "Mistake 1"
  - "Mistake 2"

clinical_pearls:
  - "Pearl 1 from High-Yield Concepts"
  - "Pearl 2"

red_flags:
  - "Red flag 1"
  - "Red flag 2"

parent_styles:
  - key: anxious
    description: "Worried parent"
    sample_lines:
      - "Quote 1"
  - key: experienced
    description: "Has dealt with this before"
    sample_lines:
      - "Quote 1"
```
